### 프로젝트 설명
본 프로젝트는 기존에 존재하는 웹서버를 조금 더 유지 보수가 편한 구조로 재개발 함으로 써, 차후 게임서버 재개발이나 어떠한 버그 또는 기능 패치를 진행할 때 불필요하게 소요되는 시간적 비용을 줄이고자 시작하게 되었습니다.

본 프로젝트는 1인 개발로 운영되고 있습니다.
